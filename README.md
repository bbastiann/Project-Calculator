# Project-Calculator

A basic calculator built with HTML, CSS and vanilla JavaScript.

## Features

- Addition, subtraction, multiplication and division
- Decimal point support
- Chained operations
- Clear (C) and Clear Entry (CE)
- Division by zero error handling

## How to use

1. Click numbers to input a value
2. Click an operator (+, -, *, /)
3. Click a second number
4. Click = to get the result
5. Click C to reset everything or CE to clear the current value

## Built with

- HTML
- CSS (Flexbox)
- JavaScript (vanilla)
